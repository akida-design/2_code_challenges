# 2_code_challenges

## Overview
This directory contains solutions to coding challenges in Python.

## Files

### count_positive_sum_negative.py
Counts the number of positive integers and sums all negative integers in an array.
- Takes an array as input
- Returns a list `[count_of_positives, sum_of_negatives]`
- Returns an empty list if the input array is empty

### merge_two_sorted_lists.py
Merges two sorted singly-linked lists into a single sorted linked list.
- Implements a `Solution` class with a `mergeTwoLists()` method
- Takes two sorted linked lists as input
- Returns a new linked list containing all elements in sorted order
